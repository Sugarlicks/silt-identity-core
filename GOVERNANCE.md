# Governance

SILT Core is a specification-first project focused on identity semantics.

## Principles

- Status before attributes  
- Authority before permissions  
- Consent before execution  
- Revocation before irreversibility  

## Stewardship

SILT Core is independently developed and versioned.

External collaboration is encouraged, including participation in standards bodies.

However, SILT Core maintains its own:

- specification direction  
- versioning  
- architectural evolution  

## Decision Making

Changes prioritise:

- semantic clarity  
- interoperability  
- long-term resilience  

Speed is secondary to correctness.
